### This console application manages products, supplier, transactions, allowing the user to add, update, delete and view details of products and suppliers with the group inventory
### It also involves Transactions, allowing user to add stock i.e., add up the quantity of a particular product by typing its id
#### Technologies used- C#, EF Core, 
#### OOAD, OOP concepts and Exception Handling.
